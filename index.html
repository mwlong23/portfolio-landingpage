Favorite Places

Everyone has a place that really makes them feel at home. Unfortunately, mine is a long flight and many miles out of reach. This inspired me to create a webpage I could visit when my happy place is out of reach. Favorite Places is a page designed to showcase all of my favorite places and remind me of why it’s so fun to visit them.

Vacation-Destination

Are you thinking about taking a vacation anytime soon? Lord knows I need one, but where would I go and what would I do. Vacation Destination is a webpage featuring a fully-loaded travel package that takes travelers out of this world. This project was built using HTML, custom CSS and Bootstrap to tie everything together.


Wiki-Clone

At Epicodus there's no denying the importance of customizing bootstrap elements using your own custom CSS stylesheet. Wiki-clone is my attempt at cloning the wikipedia hompage using HTML, my own custom CSS stylesheet and Bootstrap. This exercise demonstrated how much detail goes into each and every webpage.

Animal Shelter

If you’re looking for a furry friend and don’t know where to start this webpage is for you. Animal Shelter is a webpage that links animals in need of adoption with families who are looking for a new pet. Written in basic html and a custom CSS stylesheet, this page isn’t fancy but it provides a much needed service to pets in need.
